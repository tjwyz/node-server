# frontend-life

这是life项目

## 使用姿势

-   npm i
-   npm start


| 环境变量                     | 意义                          |
| --------------------------- | ---------------------------- |
| GEN_WEBPACK_BUNDLE_ANALYZER | 生成 bundle 组成分析           |
| USE_LOCAL_ASSETS            | 线下测试模式，静态走本地路径      |
